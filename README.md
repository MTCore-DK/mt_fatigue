# mt_fatigue

AFK & Fatigue System for FiveM (Standalone / ESX)

---

## Description

`mt_fatigue` is a lightweight and immersive AFK & fatigue system for FiveM servers.

The script tracks player inactivity and gradually applies visual fatigue effects over time.  
Fatigue is **visual only** and does not affect player control or force animations.

Supports both **standalone servers** and **ESX**, with automatic save handling.

---

## Features

- AFK detection based on player movement
- Fatigue system with smooth screen blur
- Clean and minimal NUI fatigue bar
- ox_lib notifications
- Standalone JSON saving
- ESX database saving
- Multi-language support
- Fully configurable
- Optimized and lightweight

---

## Requirements

- ox_lib
- (Optional) es_extended
- (Optional) oxmysql / mysql-async
